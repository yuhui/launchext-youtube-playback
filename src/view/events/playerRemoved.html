How this Event works

This Event is triggered when the YouTube player is removed from the web page. This can happen when:

  • The user has navigated away from the web page, for example, by going to another page, reloading the web page, or closing the browser window. When the player is removed because the user has navigated away from the web page The user's web browser could navigate away from the web page before the Rule's actions have run to completion. Unfortunately, this is the behaviour of the web browser and it cannot be controlled.
  • A script has removed the player, for example, when the user clicks a button. When the player is hidden instead of removed If the player has been hidden using CSS (for example, by setting the player's display style to "none"), then this event is not triggered. This is because the player still remains in the web page, even though the user cannot see it.

The following Data Elements are available for use with your Rule's Conditions and Actions.

Data Element Data type Value
Player State String Fixed value: player removed
Video Current Time Integer Elapsed time in seconds since the video started playing, indicated by the playhead position.
Video Duration Integer Duration in seconds of the currently playing video.
Video ID String YouTube ID of the currently loaded/playing video.
Video Loaded Fraction Floating point number Fraction of the video between 0 and 1 that the player shows as buffered.
Video Muted Boolean Whether the video is muted (true) or unmuted (false).
Video Playback Rate Floating point number Playback rate of the currently playing video.
Video Played Segment Time Integer Elapsed time in seconds since the video was last played.
Video Played Total Time Integer Total elapsed time in seconds when the video was being played.
Video Title String YouTube title of the currently loaded/playing video.
Video Type String YouTube type of the currently loaded/playing video.
Video URL String YouTube URL of the currently loaded/playing video.
Video Volume Integer The player's current volume between 0 and 100.
