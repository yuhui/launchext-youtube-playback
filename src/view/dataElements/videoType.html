How this Data Element works

This Data Element returns the YouTube type of the currently loaded/playing video.

This data element must be used with an Event from this YouTube Playback extension.

How to use:

  • %data element name%
  • _satellite.getVar('data element name', event)
When the video type is... this Data Element's value is...
Live broadcast / Live stream, Premiere live
all other video types video-on-demand
